Notes on XHTML\CSS

XHTML\CSS: -the code standards that make websites work -can view the code behind any website by right clicking in the window and selecting view source editors: -websites at basic level are text files -compiler is a web browser: all you need is a simple text editor -you can use integrated development enviroment(IDE) but this is not necessary

Notes on Tags

TAGS: -the language of XHTML revolves around tags -every component in a website is contained within a tag -the first tag you create on a website declares the doctype or the type of file your document will be

Scope: -most tags have opening and closing tags -everything in a web object is contained within the opening and closing tags -everything outside the tags belongs to another object

Head: -Head of a site includes information for browser -It is not seen by the user of the site -It has keywords for search engines, the title of the site, browser information, etc

Body: -The body includes all of the actual site content including paragraphs, images, audio, animations, backgrounds, etc

-Paragraphs: -HTML does not translate the spaces, tabs, and return characters onto the visible body elements of your site -this tag allows you to tell browser which text in web document you want to put into a paragraph form

-Inline text tags: -HTML has a tag for adding a single space called a line break(allows you to add a single space inside of a paragraph tag)

-Horizontal rule: -adds a horizontal line that acts as a separator

-Strong tag: -puts text in bold

-EM Tag: -emphasizes enclosed text by italicizing it

Notes on Other information

Other Information: -Header: the header is a system of section titles arranged from h1 to h6 in descending order. H1 has the biggest font size and H6 has the smallest

-Adding a page navigation with links: you can addi links to seperate parts of pages by naming your different sections and creating links to them

-Tool Tops: they allow the user to hover over elements of a page for further explanation of what they do

-Adding Images to your site: to adjust the size of an image, you only need to adjust the height attribute, you can also distort image by changing width
